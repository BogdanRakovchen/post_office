<b>Используемые технологии</b>
  <ul>
    <li>Spring Data Jpa</li>
    <li>Spring Web</li>
    <li>База данных: postgres</li>
    <li>Lombok</li>
    <li>Springdoc</li>
    <li>Mapstruct</li>
  </ul>
